#  删除视图

删除视图，请在视图上使用 [`db.collection.drop()`](https://www.mongodb.com/docs/v6.0/reference/method/db.collection.drop/#mongodb-method-db.collection.drop) 方法。

 例如，这个命令删除一个名为`productView01`的视图:

```javascript
db.productView01.drop()
```
